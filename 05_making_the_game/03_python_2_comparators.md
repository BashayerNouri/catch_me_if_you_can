

In python there are several ways to compare between two values. The result of the comparison is a boolean which is either  `True`  or  `False`.

-   Equal to  `==`
-   Not equal to  `!=`
-   Less than  `<`.
-   Less than or equal  `<=`.
-   Greater than  `>`.
-   Greater than or equal  `>=`.

---
***Example:***

```python
print(2>6)
print(5==5)
print(2!=10)
```

***Output:***
```
>>> False
>>> True
>>> True
```
